🚀GoTogether – A smart student commute coordination system.


👥 Team – Next-Gen AI & ML
Team Leader: Kaushal Kumar Ray
Members: Adarsh Sanjeet Kumar Singh, Raju Kumar

🔹 Key Features

    Ride creation & search
    Join request approval system
    Ride-based chat
    Leave ride with reason notification
    AI Route Optimizer (OpenRouteService API)
    SOS emergency module
    OTP-based authentication

⚙️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
API: OpenRouteService

▶️ Run Locally
pip install -r requirements.txt
python app.py

Open in browser:
http://127.0.0.1:3000
